# 2026新年祝福网站

这是一个为2026年新年创建的跨年祝福语生成器网站，提供各种新年祝福功能。

## 功能特性

### 🎯 核心功能
- **实时倒计时**：精确显示距离2026年还有多少时间
- **祝福语生成**：根据不同类型生成个性化新年祝福
- **专属祝福**：根据选择的愿望生成专属祝福语
- **祝福语库**：精选的2026年跨年祝福语
- **新年决心**：制定2026年目标和计划
- **烟花表演**：绚丽的2026年烟花动画效果

### 🎨 设计特色
- **响应式设计**：适配各种屏幕尺寸
- **PWA支持**：可安装到桌面，支持离线使用
- **交互丰富**：动画效果、音效、通知等
- **主题定制**：深色主题，新年氛围浓厚

### 🔧 技术特点
- **纯前端实现**：无需后端服务
- **模块化代码**：易于维护和扩展
- **本地存储**：祝福语和设置自动保存
- **服务工作者**：缓存资源，提升加载速度
